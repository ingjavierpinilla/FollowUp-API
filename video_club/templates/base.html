Recibe el ID de la sucursal de la cual se desea la informacion, retorna un json con la suma de las ventas por mes con la fecha en formato ISO 8601 sin incluir la hora, minutos y segundos

  • URL

    /api/sucursal/<id>

  • Method:

    GET

  • URL Params

    Required:

    id=[integer]

  • Success Response:

    • Code: 200
    • Content:
      [
      {"mes": "2021-01-01T00:00:00Z",
      "valor_venta": 40.0 }
      { "mes": "2021-02-01T00:00:00Z",
      "valor_venta": 60.0 }
      ]

  • Error Response:

    • Code: 401 UNAUTHORIZED

    OR

    • Code: 400 BAD REQUEST
      Content: {'ID invalido.'}

    OR

    • Code: 204 NO CONTENT
      Content: {'Cinta <id> no encontrada.'}

Recibe una fecha en formato ISO 8601 sin incluir la hora, minutos y segundos i.e. 2010-12-16

Retorna un JSON con codigo_sucursal, cintas_alquiladas y valor_venta. Ordenado por el codigo de sucursal

  • URL

    /api/venta/?fecha

  • Method:

    GET

  • URL Params

    Required:

    fecha=[string] fecha en formato ISO 8601 i.e. 2010-12-16

  • Success Response:

    • Code: 200
      Content:
      [
      { "codigo_sucursal": 1,
      "cintas_alquiladas": 2,
      "valor_venta": 40.0 },
      { "codigo_sucursal": 2,
      "cintas_alquiladas": 1,
      "valor_venta": 20.0 }
      ]

  • Error Response:

    • Code: 401 UNAUTHORIZED

    OR

    • Code: 400 NOT FOUND
      Content: {'Fecha no valida'}

    OR

    • Code: 204 NO CONTENT
      Content: {'Sin informacion para la fecha requerida.'}

  • URL

    /api/cinta/?disponible=true

  • Method:

    GET

  • URL Params

    Required:

    None

  • Success Response:

    • Code: 200
      Content: [
      { "id": 1,
      "titulo": "titulo1",
      "valor": 10.0,
      "disponible": true },
      { "id": 2,
      "titulo": "titulo2", "valor": 20.0,
      "disponible": true },
      { "id": 3,
      "titulo": "titulo3",
      "valor": 30.0,
      "disponible": true }
      ]

  • Error Response:

    • Code: 401 UNAUTHORIZED
    • Code: 400 BAD REQUEST

Recibe dos fechas (de, hasta) en formato ISO 8601 sin incluir la hora, minutos y segundos i.e. 2010-12-16

Retorna un JSON con codigo_sucursal, codigo_sucursal__nombre y valor_venta. Ordenado por el codigo de sucursal

  • URL

    /api/venta/?de&hasta

  • Method:

    GET

  • URL Params

    Required:

    de=[string] fecha en formato ISO 8601 i.e. 2010-12-16 hasta=[string] fecha en formato ISO 8601 i.e. 2010-12-16

  • Success Response:

    • Code: 200
      Content:
      [
      { "codigo_sucursal": 1,
      "codigo_sucursal__nombre": "s1",
      "valor_venta": 40.0 }
      ]

  • Error Response:

    • Code: 401 UNAUTHORIZED

    OR

    • Code: 400 BAD REQUEST
      Content: {'Fecha no valida'}

    OR

    • Code: 400 BAD REQUEST
      Content: {'Hasta' ocurre antes que 'de'}

    OR

    • Code: 204 NO CONTENT
      Content: {'Sin informacion para la fecha requerida.'}
